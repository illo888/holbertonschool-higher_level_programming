# Python - Exceptions

This project covers:
- Errors vs Exceptions
- How to use try / except / finally
- Raising built-in exceptions
- Printing errors to stderr
- Safe execution of functions
- Cleanup actions after exceptions

## Requirements
- Ubuntu 20.04, Python 3.8.5
- pycodestyle 2.7.*
- All files start with #!/usr/bin/python3
- All files executable
- No external modules allowed
