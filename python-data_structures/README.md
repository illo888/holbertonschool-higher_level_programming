## Tasks üìÉ

### 0. Print a list of integers
**File:** `0-print_list_integer.py`  
Python function that prints all integers of a list, one per line.  
- Without importing modules or casting integers into strings.

---

### 1. Secure access to an element in a list
**File:** `1-element_at.py`  
Python function that retrieves an element from a list.  
- If `idx` is negative or out of range, returns `None`.  
- Without importing modules or using `try/except`.

---

### 2. Replace element
**File:** `2-replace_in_list.py`  
Replaces an element of a list at a specific position.  
- If `idx` is negative or out of range, returns the original list.  
- Without importing modules or using `try/except`.

---

### 3. Print a list of integers... in reverse!
**File:** `3-print_reversed_list_integer.py`  
Prints all integers of a list in reverse order.  
- Without importing modules or casting integers into strings.

---

### 4. Replace in a copy
**File:** `4-new_in_list.py`  
Replaces an element of a list at a specific position without modifying the original list.  
- If `idx` is negative or out of range, returns the original list.  
- Without importing modules or using `try/except`.

---

### 5. Can you C me now?
**File:** `5-no_c.py`  
Removes all characters `c` and `C` from a string and returns the string.  
- Without importing modules or using `str.replace()`.

---

### 6. Lists of lists = Matrix
**File:** `6-print_matrix_integer.py`  
Prints a matrix of integers, one row per line.  
- Without casting integers into strings.

---

### 7. Tuples addition
**File:** `7-add_tuple.py`  
Adds two tuples and returns a tuple with two integers:  
- First element: sum of first elements.  
- Second element: sum of second elements.  
- Missing values replaced with `0`.  
- Without importing modules.

---

### 8. More returns!
**File:** `8-multiple_returns.py`  
Returns a tuple with the length of a string and its first character.  
- If string is empty, first character is `None`.  
- Without importing modules.

---

### 9. Find the max
**File:** `9-max_integer.py`  
Finds the biggest integer of a list.  
- If list is empty, returns `None`.  
- Without importing modules or using `max()`.

---

### 10. Only by 2
**File:** `10-divisible_by_2.py`  
Finds all multiples of 2 in a list.  
- Returns a new list of booleans (`True` or `False`).  
- Without importing modules.

---

### 11. Delete at
**File:** `11-delete_at.py`  
Deletes an item at a specific position in a list.  
- If `idx` is negative or out of range, returns the original list.  
- Without importing modules or using `pop()`.

---
